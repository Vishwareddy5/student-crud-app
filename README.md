# Student Management CRUD App

A full-stack **CRUD** application to manage student records using:

- **Node.js**
- **Express.js**
- **MongoDB (with Mongoose)**
- **EJS Templating Engine**

This app allows you to **Create, Read, Update, and Delete** student records.
